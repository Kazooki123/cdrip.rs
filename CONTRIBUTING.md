# Contributing to CDRip.rs

We'd appreciate your contribution! CDRip.rs goal is to make **CD Archiving** easier and simple for hobbyist and genuine archivers, the project may look polished, however there are still some features and built-ins that need to be improved (e.g `paths`, `metadatas`, etc).

Here's how you can contribute! 💿💖

## Fork

Fork the repository first, then **git clone** it to your terminal.

```bash
git clone https://codeberg.org/username/cdrip.rs # codeberg
git clone https://github.com/username/cdrip.rs   # github
```

It's up to you on which git services to choose from, though i recommend **Codeberg** for now..

## Feature

If you'd want to add a feature or a major change to the ripper, **please clarify** the description of your feature and HOW it can be useful for the software tool.

```bash
git branch my-feature
git checkout -b my-feature

git commit -m "Your feature description"
```

## Pull Requests

Once done, you can then make a Pull Request **(PR)**, please note that the review may take hours or days, so be patience, thanks!

You should include a:

- Clear description
- What it does
- How it helps the software
- How **efficient** is it
